Coloca aquí tus imágenes de productos.

Formato esperado (por defecto):
- Para cualquier item (ventas o coleccionables): usa el nombre `item-<ID>.jpg`
  Ejemplos:
    item-1.jpg         -> para "Juego 1"
    item-23.jpg        -> para "Juego 23"
    item-1001.jpg      -> para "Coleccionable 1" (IDs de coleccionables comienzan en 1001)
    item-1037.jpg      -> último coleccionable

Puedes usar .png o .webp también. Si usas otra extensión, actualiza la ruta en data.js (propiedad `img`).

Si no agregas imagen, el cuadro mostrará solo el fondo degradado.
